# enrichr
Annotate, compute, and visualize fold-changes and Z-scores for differential proteomics results.

### Functions
`enrich()`
`psmplot()`
`combine_reps()`
`zplot()`

### Installation
Run the following code:
``` r
devtools::install_github("rachaelcox/enrichr")
library(enrichr)
```
